Noticioso

Mobile  ·  RSS Reader  ·  Open Source

A personal RSS reader with full article extraction, built for e-ink devices. No algorithm, no noise, just the articles you chose to follow.

Download APK

Features

  • Feed discovery — paste a URL or type a search term. The app queries DuckDuckGo, visits each result, and auto-discovers RSS/Atom feeds.
  • Full article extraction — fetches the original article and runs it through Mozilla Readability for a distraction-free reading experience.
  • Paginated reader — all content renders inside a WebView using CSS multi-column layout. Swipe left/right to flip pages, swipe up/down to navigate between screens.
  • Share to read — share any URL from your browser directly into Noticioso to read it as a clean article.
  • Import / Export — back up and restore your feed list as a JSON file via the native share sheet and document picker.
